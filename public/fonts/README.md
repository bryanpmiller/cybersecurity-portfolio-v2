# Local Portfolio Fonts

This folder contains self-hosted web font files used by the portfolio:

- `inter-variable.woff2`
- `space-grotesk-variable.woff2`
- `jetbrains-mono-regular.woff2`
- `jetbrains-mono-medium.woff2`
- `jetbrains-mono-semibold.woff2`
- `jetbrains-mono-bold.woff2`

The matching OFL license files are kept here with the downloaded fonts.
